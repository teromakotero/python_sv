Lista

En lista är ett verktyg för att lagra en stor mängd information samtidigt. Den kan definieras som en variabel med andra variabler inuti. Listan består av antal nummer-objekt-par. Varje objekt kan nås med ett nummer (räkningen startar från noll).

Exempel på en lista i Python

En lista är lätt att skapa. Först ges ett namn och sedan värdena inom parentes. Nedan skapas en tom lista som heter “lista”.

lista = []

Använd det här kommandot för att lägga till ett objekt (i detta fall heltalet 8) i slutet av listan.

lista.append(8)

Detta kommando raderar och återställer det sista objektet.

lista.pop()

Detta kommando placerar ett objekt på en viss plats i listan.

lista.insert(plats,objekt)

Använd det här kommandot för att ta bort ett visst objekt från listan.

lista.remove(objekt)

Använd det här kommandot för att välja ett objekt på en viss plats i listan.

lista.index(plats)

Använd det här kommandot för att ta reda på listans längd.

len(lista)

Detta kommando berättar hur många gånger ett objekt visas i listan.

lista.count(objekt)
