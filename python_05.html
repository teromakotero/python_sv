Villkorssats

Inom programmering används villkorssatser för att programmet ska fungera som man vill när villkoren uppfylls. Villkoren kan antingen uppfyllas (true) eller inte (false). Om ett villkor uppfylls, händer något. Villkorssatser kan alltså användas för att förgrena programmets utförande. Villkorssatser kan man också lägga in i varandra och på så sätt göra mer komplexa förgreningar. Det finns olika möjligheter i Python att skapa villkorssatser.

Exempel på villkorssatser i Python

Denna if-villkorssats kör de indenterade satserna under den, om villkoret uppfylls.

       if något villkor:
            så händer något

Denna if else-villkorssats kör de indenterade satserna under if, om villkoret uppfylls. Annars kör den de indenterade satserna under else.

       if något villkor:
            så händer något
        else:
            så händer något annat

Om det finns fler villkor, görs det i Python såhär (elif är en förkortning av else if).

       if något villkor:
                så händer något
        elif andra villkoret:
                så händer något annat
        elif etc. villkoret:
                så händer något annat
        else:
                så händer något annat
