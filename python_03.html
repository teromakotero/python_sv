Slinga

Inom programmering kan man använda slingor för att upprepa samma satser eller kodblock många gånger. I Python finns olika möjligheter att skapa slingor.

Exempel på slingor i Python

Den oändliga while-slingan

Denna slinga upprepar den indenterade koden under den i all oändlighet eller tills programmet avlutas.

while True:

Efter while kan man ställa in ett villkor. Upprepningen fortsätter så länge som villkoret är sant.

for-slinga

Denna slinga upprepar den indenterade koden under den ett visst antal gånger. I detta fall upprepas kommandona inom slingan tio gånger.

for i in range(10):
